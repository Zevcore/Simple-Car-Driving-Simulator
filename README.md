# Car Simulator

Simple Car Simulator in Window (Java)

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Installing

Clone this repository and compile it using jdk. You can also deploy it by your IDE.

```
git clone https://github.com/Zevcore/car-simulator
```
